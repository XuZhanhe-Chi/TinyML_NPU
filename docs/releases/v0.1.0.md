# TinyML_NPU v0.1.0

TinyML_NPU v0.1.0 是首个面向研究和教学的公开版本。它提供一条可检查的 INT8 TinyML NPU 链路：受约束的 ONNX QDQ compiler、32-byte VenusCore uOP、SpinalHDL RTL、Zynq bare-metal runtime，以及在原版 Digilent Zybo XC7Z010 上验证的 KWS testvector 闭环。

## Release Highlights

- 1 cluster、4 lanes、SIMD4 的 VenusCore INT8/INT32 数据通路。
- APB3 control、AHB-Lite DMA 和 128 KiB shared BRAM integration。
- 44-uOP KWS bundle，PS runtime relocation 和结构化 JTAG result ABI。
- Python compiler tests、behavioral simulator、glue RTL protocol regressions 和 GitHub Actions。
- Vivado/Vitis 2021.1 非交互构建与实体板自动验收。
- 面向学习者的架构、compiler、ISA、runtime、bring-up 和验证文档。

| KWS uOP count | 44 |
| top1 | 0，expected 0 |
| INT8 byte mismatches | 10 / 12 |
| Maximum absolute error | 5，tolerance 5 |
| NPU active cycles | 563,188 |
| Active-cycle equivalent | 11.26376 ms @ 50 MHz |
| Post-route WNS / WHS | +3.025 ns / +0.019 ns |
| Unrouted / partial nets | 0 / 0 |

11.26376 ms 只由 NPU active-cycle counter 换算，不包含 PS 数据准备、cache、固件启动、UART 或 JTAG 开销，不是端到端延迟或吞吐率。

## Assets

- `tinyml_npu_zybo7010_v0.1.0.bit`：可直接下载到 PL 的 bitstream。
- `tinyml_npu_zybo7010_v0.1.0.xsa`：包含 bitstream 的 Vivado hardware platform。
- `tinyml_npu_kws_test_v0.1.0.elf`：bare-metal KWS testvector application。
- `MANIFEST.json`、`SHA256SUMS` 和 `VERIFICATION.txt`：来源、校验和与验收摘要。

这些二进制由 AMD Xilinx Vivado/Vitis 2021.1 生成，可能包含受 vendor terms 约束的生成内容。Apache-2.0 适用于仓库原创源码，不覆盖第三方和 vendor 权利。下载或再分发前请阅读 `THIRD_PARTY_NOTICES.md`。

## Known Boundaries

- 仅支持原版 Digilent Zybo XC7Z010，不支持 Zybo Z7。
- 仓库不分发模型、训练代码或数据集。
- 演示只使用固定 testvector，没有实时音频输入。
- Compiler 是受约束的研究实现，不是通用 ONNX runtime。
- GitHub CI 不运行专有 Vivado/Vitis 或实体板测试；本 Release 的 manifest 记录人工发布验收。

## English Summary

v0.1.0 is the first research and teaching release of TinyML_NPU. It publishes an inspectable compiler-to-FPGA path and a physically validated KWS testvector demo on the original Digilent Zybo XC7Z010. The release passes 30 compiler tests, protocol-level glue RTL regressions, 50 MHz implementation timing, and the structured physical-board acceptance test. Generated vendor artifacts carry separate notices.
